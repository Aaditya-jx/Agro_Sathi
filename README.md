# AgroSathi – Sustainable Agriculture Advisory System

🌱 A comprehensive platform for farmers promoting sustainable agriculture practices with AI-powered disease detection, crop advisory, and eco-friendly solutions.

## 🏗 Architecture

- **Frontend Web**: React.js (Vite) + TailwindCSS
- **Mobile App**: React Native (Expo) + NativeWind
- **Backend**: Firebase (Auth, Firestore, Storage, FCM)
- **AI**: TensorFlow.js (Web) + TensorFlow Lite (Mobile)
- **State Management**: Context API / Redux Toolkit

## 📂 Project Structure

```
agrosathi/
├── web/           # React Web App
├── mobile/        # React Native App
├── firebase/      # Firebase config & rules
├── shared/        # Shared utilities & types
└── README.md
```

## 🚀 Quick Start

### Web App
```bash
cd web
npm install
npm run dev
```

### Mobile App
```bash
cd mobile
npm install
npx expo start
```

### Firebase Setup
1. Create Firebase project
2. Configure Firestore rules
3. Set up Authentication
4. Enable Storage and FCM
